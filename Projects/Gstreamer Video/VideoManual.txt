1) Чтобы использовать модуль видео надо импортировать его:
    	import RTCVideo

2) Видео передается через rtp поток, соответственно нужно указать ip(передающего устройства) и порты для связи:
    	IP = '127.0.0.1'
	RTP_RECV_PORT0 = 5000
	RTCP_RECV_PORT0 = 5001
	RTCP_SEND_PORT0 = 5005

3) Чтобы все заработало, нужно создать экземпляр класса Video, указав ему порты и ip
	video=RTCvideo.Video(IP, RTP_RECV_PORT0, RTCP_RECV_PORT0, RTCP_SEND_PORT0)
Тут еще есть скрытый параметр overlay, который позволяет создать свою drawcallback - функцию для отрисовки чего-нибудь поверх экрана(лучше не пробовать, если не знаешь )

4) Чтобы нарисовать текстуру (просто поверх экрана), нужно вызвать метод drawOverlay
	video.drawOverlay("ring.png", x = 0, y = 0, scaleX = 0.3, scaleY = 0.2, fullScreen = True)

Первый параметр - путь до изображения, которое нужно наложить
x и y - координаты изображения на экране
scaleX и scaleY  - масштабирующие параментры, на которые умножаются координаты(соответственно уменьщается изображение)
fullScreen - в положении True растягивает изображение на весь экран, игнорируя предыдущие 4 параметра.

5) Чтобы запустить прием видео, необходимо вызвать:
	video.start()

6) Чтобы поставить видео на паузу, необходимо вызвать:
	video.paused()

7) Чтобы остановить видео и освободить все ресурсы, необходимо вызвать:
	video.stop()
